# os-practical-sessions
OS Practical Assignments: Mutex Locks, Producer-Consumer (Semaphores), Counting Semaphores, and Shared Memory IPC in C.
"This repository contains solutions for 4 practical sessions on Operating Systems:

1. Mutex Lock Demonstration
2. Producer-Consumer Simulation using POSIX Semaphores
3. Counting Semaphore Implementation
4. Shared Memory Programming with IPC

Compile with -pthread and (for practical 4) -lrt."
